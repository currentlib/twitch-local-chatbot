Список модів:

reply - просто відповідає на повідомлення заданою фразою
параметри:
    1) mode - reply
    2) triggers - список повідомлень на які буде відповідати бот
    3) message - повідомлення яке надішле бот

replyAny - відповідає на будь-яке повідомлення яке містить хоча б одну з перелічених фраз
параметри:
    1) mode - replyAny
    2) triggers - список фраз на які буде реагувати бот в повідомленнях
    3) message - повідомлення яке надішле бот

replyCommand - відповідає на команду заданою фразою
    1) mode - replyCommand
    2) triggers - список повідомлень на які буде відповідати бот
    3) message - повідомлення яке надішле бот

random - функція з випадковими числами
    1) mode - random
    2) triggers - список команд на які буде реагувати бот і виконувати певну функцію
    3) message - повідомлення яке  надішле бот з заміною ключових слів
    4) isParam - чи потрібен параметр для виконання команди
    5) isBool - чи рандом буде бінарний з заміною так/ні на
    6) range - мінімальне значення рандому
